# Контрольный пример: неясный объект

## Вход

На изображении видны размытые геометрические формы без распознаваемой ходовой части, кабины, кузова или рабочей части. Назначение объекта определить нельзя.

## Ожидаемое поведение

- `source_type`: непонятный технический объект
- уровень уверенности: низкая
- общий класс техники не определяется надёжно
- навык сообщает, каких признаков не хватает
- навык должен запросить дополнительный ракурс, более чёткое изображение или текстовое уточнение
- полный анализ карточки не выдумывается
- раздел «Навык применён полностью» не подтверждается
- данные не фиксируются
- генерация изображения не начинается
