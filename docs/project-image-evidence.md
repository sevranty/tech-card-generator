## Визуальное review

Проверено:

- изображена одна гусеничная машина;
- рабочий щит связан с корпусом и касается грунта;
- видимый грунт перед щитом показывает рабочее действие;
- отсутствуют люди, животные и вторая техника;
- отсутствуют бренды, модели, логотипы и номера;
- project image не имитирует интерфейс ChatGPT, Codex, GitHub или dashboard;
- композиция содержит визуальную структуру карточки 2×1;
- справа показана только отдельная рабочая часть, а не вторая машина;
- фон легче и проще машины;
- внешний safe zone соблюдён;
- ключевые элементы читаются при уменьшении до 320×160;
- SVG master использует русские продуктовые тексты;
- переносимый PNG-builder использует только контролируемые ASCII-подписи и не создаёт псевдотекст.

## Ограничение среды

Репозиторий содержит master и воспроизводимый builder. Фактическая загрузка `assets/social-preview.png` в поле GitHub **Social preview** относится к настройке репозитория и не может быть подтверждена одним только commit/PR. До проверки настройки нельзя утверждать, что GitHub уже использует новый preview во внешних карточках ссылок.
