# TCG#8 — project image и GitHub social preview

## Назначение

Визуал представляет `machinery-card-generator`, а не интерфейс, dashboard или статусный отчёт. Главный образ — одна реальная механически правдоподобная машина, выполняющая понятную рабочую задачу, рядом с отдельным изображением её рабочей части.

## Концепция

- чёрно-белая техническая линейная иллюстрация;
- одна гусеничная машина без бренда, модели и регистрационных обозначений;
- видимый контакт рабочего щита с грунтом;
- структура образовательной карточки 2×1 из двух равных квадратных блоков;
- левый блок показывает действие, правый — отдельную рабочую часть;
- никаких людей, животных, второй техники, логотипов или псевдоинтерфейсов;
- фон заметен, но светлее и проще машины;
- локальная белая зона отделяет машину от линий окружения без свечения и тени.

## Артефакты

- `assets/project-image.svg` — редактируемый master 1600×800;
- `assets/project-image.png` — воспроизводимый project image 1600×800;
- `assets/social-preview.png` — воспроизводимый GitHub social preview 1280×640;
- `scripts/build-project-image.py` — builder и размерный gate.

SVG — редактируемый источник композиции. PNG-файлы генерируются builder-скриптом и могут быть воспроизведены без изменения смысла визуала.

## Safe zones

- внешний безопасный отступ — 10% от меньшей стороны;
- название проекта и ключевой смысл находятся в левой центральной зоне;
- карточка 2×1 находится в правой центральной зоне;
- критичные элементы не касаются краёв и сохраняются при уменьшении;
- композиция проверяется в полном размере и на миниатюре 320×160.

## Размеры и формат

| Артефакт | Размер | Формат | Ограничение |
|---|---:|---|---|
| project image | 1600×800 | PNG, монохромный | менее 1 MB |
| GitHub social preview | 1280×640 | PNG, монохромный | менее 1 MB |
| master | 1600×800 | SVG | редактируемый источник |

## Генерация

```bash
python3 scripts/build-project-image.py
```

Успешный запуск должен закончиться строкой:

```text
Project image validation passed
```

## GitHub repository setting

Для фактического отображения social preview в карточках ссылок необходимо загрузить `assets/social-preview.png` через настройки репозитория GitHub. Наличие файла в дереве и фактическая настройка GitHub — разные проверки.
